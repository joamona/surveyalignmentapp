Application to get distances to an alineation defined by two points.

This program allows the surveyor to stakeout alineations parallel, or perpendicular to a reference line defined by two points (pt1 and pt2).

Calculates the azimuth and length of the reference line (pt1-pt2).

Calculates the intersection (pt4) of the reference line with a perpendicular line passing by pt3.

Calculates perpendicular distance of the reference line to pt3 (length of pt3-pt4).

Calculates the distance from the origin (length of pt1-pt4).

You can also set offsets to add or substract to the calculated distances
