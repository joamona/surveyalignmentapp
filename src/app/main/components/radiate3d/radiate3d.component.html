Fórmulas utilizadas:

  • Azimut = Lectura Horizontal + Desorientación
  • Distancia Reducida (Dr) = Distancia Geométrica * cos(Lectura Vertical)
  • t = Distancia Geométrica * sin(Lectura Vertical)
  • Coeficiente de refracción (Cr) = ((1 - K) / (2 * 6371000)) * Dr²
  • X_radiado = X_base + Dr * sin(Azimut)
  • Y_radiado = Y_base + Dr * cos(Azimut)
  • Z_radiado = Z_base + t + i - m + Cr

La fórmula para la corrección por refracción (Cr) es: (1 - K) / (2 * RadioTierra) * Dr², donde el Radio de la Tierra se toma como 6.371.000 metros. Por defecto para España el coeficiente es K = 0.08. Si se quiere anular esta corrección en el cálculo, solo hay que usar K = 1.
